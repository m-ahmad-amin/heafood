# heafood
HeaFood helps students and calories-conscious users find quick and healthy meals based on their input calories range.

# How to run
1. Go to the [Spoonacular API](https://spoonacular.com/food-api) website.  
2. Sign up for a free account and get your **API key**.  
3. Open the `index.html` file in your project.  
4. Replace `YOUR_SPOONACULAR_API_KEY` in the `<script>` section with your API key.  
5. Open the HTML file in your browser to **start finding recipes**.
